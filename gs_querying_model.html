Querying modelÂ¶

Each deployed on cluster model can be queried using HTTP Api (if EDI and Ingress are enabled in configuration).

To perform this call, JWT token, generated by EDI service, is required (if it is enabled in configuration). For details see chapter.

Models can be queries using next mechanisms:

  • legionctl CLI tool. See chapter
  • EdgeClient in Python SDK (not in toolchain)
  • Plain HTTP calls. See chapter
