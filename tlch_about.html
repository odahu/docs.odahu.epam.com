About toolchains¶

Toolchains are SDKs for programming languages that should be used for developing ML models that are compatible with Legion.

Supported toolchains¶

Legion supports next toolchains:

  • Legion’s Python toolchain - SDK for Python language that allows to export Python functions as a models. It support ML frameworks such as scikit-learn.
