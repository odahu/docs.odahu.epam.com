Development¶

Pre requirements¶

To participate in developing of Odahu project you have to meet these requirements:

  • Linux/macOS operating systems (due to tools used for development)
  • Python:
    • Python v3.6
    • Pipenv
  • Golang:
    • Golang v1.12
    • Dep
    • golangci-lint
    • Kubebuilder
    • swag
    • gotestsum
  • Jupiterlab plugin:
    • Typescript
    • Yarn
  • Infra:
    • HELM v2.14.0
    • Kubectl v1.13.2
    • Docker v17+
    • Swagger codegen 2.4.7
