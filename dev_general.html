Development¶

Pre requirements¶

To participate in developing of Odahu project you have to meet these requirements:

  • Linux/macOS operating systems (due to tools used for development)
  • Python:
    • Python v3.6
    • Pipenv
  • Golang:
    • Golang v1.14
    • Dep
    • golangci-lint
    • Kubebuilder
    • swag
    • gotestsum
  • JupyterLab plugin:
    • Typescript
    • Yarn
  • Infra:
    • HELM v3.2.4
    • Kubectl v1.16.10
    • Docker v17+
    • Swagger codegen 2.4.7
